This project deals with data cleaning of global layoffs information done in MySQL. Data Cleaning procedure involves removing duplicate entries that exist in the data set, standardizing the data, checking whether it is possible to populate null or blank values utilizing the information provided in the data set and lastly, removing any columns that are not needed.

SQL skills utilized in this project:
-Importing process
-Creating databases
-Creating Tables
-Subqueries & partitioning
-Deleting, Altering & Updating
-WHERE statements
-Joins, specifically Self Join

Open MySQL, create a Database, create a Table via Table Data Import Wizard.
Remember to tick 'Drop table if exists' box. DO NOT CONFIGURE DATA WHEN ASKED DURING THE IMPORT PROCESS.

Refer to the comments in the code.
